# Internship-Task-1

Population Data Visualization
Overview

This project analyzes global population data and visualizes the top 10 most populated countries in 2022 using Python, Pandas, and Matplotlib.

The dataset is processed to extract population values, clean missing data, and generate a bar chart showing the countries with the highest populations.

Features
Load and process population data from a CSV file.
Clean and preprocess the dataset.
Extract 2022 population statistics.
Identify the top 10 most populated countries.
Create a bar chart for easy visualization.
Technologies Used
Python
Pandas
Matplotlib

Installation
Clone the repository:

Usage
Ensure the population dataset CSV file is present in the project directory.
Open and run the Jupyter Notebook:
jupyter notebook Task_1.ipynb
Execute all cells to generate the population visualization.
Output

The notebook generates a bar chart displaying the Top 10 Most Populated Countries (2022).

Dataset

The project uses World Bank population data containing country-wise population statistics.
